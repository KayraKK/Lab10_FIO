# Lab10_FIO

Проект проверки данных пользователя при регистрации.

## Требования
- Логин: телефон (+X-XXX-XXX-XXXX), email (xxx@xxx.xxx) или строка (мин 5 символов, латиница, цифры, _)
- Логин не должен быть в черном списке (admin, root, user, test, system, moderator)
- Пароль: мин 7 символов, кириллица, цифры, спецсимволы, есть заглавная и строчная буква, цифра и спецсимвол
- Пароль и подтверждение совпадают
